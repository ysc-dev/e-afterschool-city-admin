# 학교 외의 방과 후 관리 프로그램 - 관리자 버전

경남 지역 교육청을 대상으로 프로젝트를 진행

## 기능
- 관리자로 로그인
- 대시보드
- 안내장 관리
- 학교 관리
- 강사 관리
- 학생 관리
  - 학생 조회
  - 수강 이력 조회
- 과목 관리
  - 과목 그룹 관리
  - 수강 과목 조회
  - 수강 과목 등록
- 수강 관리
  - 수강 신청 조회
  - 수강 대기 조회
  - 수강 취소 조회
- 횟수별 수업 관리
- 공지사항
- 만족도 및 설문 관리
- SMS 전송
- 사용자 정보

## 기술
- Spring Boot
- JPA
- JSP
- MySQL
- Maven
- Tiles
